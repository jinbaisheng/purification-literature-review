一、研究背景与创新点#

1.1 领域概述#

抗体药物的疗效很大程度上取决于其效应功能，其中 ADCC（抗体依赖的细胞介导的细胞毒性作用） 是肿瘤免疫治疗的关键机制之一。Fc 段与 FcγR IIIa（CD16）的结合强度直接决定了 ADCC 活性的高低。

1.2 现有方法的痛点#

传统 IgG1 抗体的 ADCC 活性存在以下局限：

  • FcγR IIIa 亲和力有限：野生型 IgG1 与 FcγR IIIa 的结合较弱
  • 糖基化依赖性强：ADCC 活性高度依赖 Fc 糖链的岩藻糖修饰
  • 批间差异大：细胞培养条件影响糖基化，导致活性不稳定

1.3 本文创新点#

  1. 定点突变技术：引入 F243L/N434W 等突变位点，增强 FcγR IIIa 亲和力
  2. 去岩藻糖化修饰：通过 CHO 细胞基因工程实现完全去岩藻糖化
  3. 联合突变策略：结合糖工程与氨基酸突变，实现 ADCC 活性最大化
