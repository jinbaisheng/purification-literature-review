抗体电荷异构体：表征与控制策略#

本资料详细介绍了抗体电荷异构体的形成机制、表征方法及控制策略。

  1. 电荷异构体概述

  4. C端赖氨酸剪切
  5. 氧化和糖化
  6. 控制策略

1. 电荷异构体概述#

电荷异构体是单克隆抗体的关键质量属性（CQA）。

关键要点#

  • 电荷变化可能影响药物稳定性
  • 可能导致免疫原性问题
  • 需要全面的表征技术

2. 酸性峰和碱性峰定义#

Ion Exchange Chromatography (IEX) 分析#

类型 描述 pI 变化
酸性峰 主峰前洗脱 较低 pI
主峰 天然亚型 正常 pI
碱性峰 主峰后洗脱 较高 pI

3. 脱酰胺和糖基化#

脱酰胺 (Asn → Asp)#

  • Asn 残基的自发脱酰胺
  • 序列基序：Asn-Gly, Asn-Ser
  • 碱性条件下加速
  • 导致酸性偏移

唾液酸化#

  • 添加唾液酸残基
  • 增加负电荷
  • 细胞培养条件影响唾液酸化水平

4. C端赖氨酸剪切#

机制#

  • 羧肽酶活性去除 C 端 Lys
  • 不完全加工导致 +1, +2 Lys 变体
  • 在 IEX 色谱碱性区域更明显

控制策略#

  • 优化细胞培养条件
  • 使用羧肽酶抑制剂
  • 考虑纯化过程中的酶处理

5. 氧化和糖化#

甲硫氨酸氧化#

  • Met 残基氧化（尤其是 Met252, Met428）
  • 导致酸性偏移
  • 光、金属和过氧化物加速

糖化#

  • 葡萄糖非酶促附着到 Lys 残基
  • 导致碱性变体形成
  • 通过培养基优化控制

6. 控制策略#

过程控制要点#

  1. 细胞培养：pH、温度、持续时间
  2. 纯化：捕获、精制、病毒灭活
  3. 制剂：缓冲液组成、pH 稳定性

分析监测#

  • IEX-HPLC：电荷变体分析
  • CEX：酸/碱分离
  • cIEF：等电点测定
